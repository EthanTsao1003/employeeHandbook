###中学考试题
1. （6、7）二人分栈合、二人分队合（txt、word、pdf）。
提升难度要求：手动解决冲突，git能解决大部分的冲突，这些情况得手动解决冲突，不知道1在前还是2在前，git没有那么智能。
主分支代码，，起床，睡觉。
分支1    起床、程序员1写代码，睡觉（合并）。
分支2    起床、程序员2写代码，睡觉（合并，会有冲突，需要手动解决冲突）。
手动改成  起床，1写代码，2写代码（顺序无所谓，但是都得写代码），睡觉。
